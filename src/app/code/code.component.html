API Props

buttonType

Determines the button's HTML type

  • button (default)
  • submit
  • reset

type

Determines what type appears on the button

  • primary (default)
  • secondary
  • destroy
  • confirm
  • subtle
  • cancel

size

Determines the button's size

  • small
  • medium (default)
  • large

status

Adds a status indicator to the button

  • spinning - Adds a spinner next to the text
  • success - Makes the button green and adds a checkmark next to the text
  • failure - Makes the button red and adds a X next to the text
